# Kona FC Website

A fully responsive, multi-page website for Kona FC, a fictional football club based in Nairobi, Kenya.
Built with HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

## Pages
1. Home — Hero section, latest results, next match, player spotlight
2. About — Club history, stats, mission, location
3. Fixtures & Results — Season summary, full results table
4. News — Articles with live search filter
5. Membership — Fan, Supporter and Platinum plans
6. Shop — Club merchandise with order links
7. Contact — Contact form with JS validation, contact info
8. Help — FAQ section

## Technologies Used
- HTML5 (semantic tags)
- CSS3 (custom properties, flexbox, media queries)
- JavaScript ES6+ (form validation, live search)
- Bootstrap 5.3.3 (grid, navbar, responsive components)
- Git & GitHub (version control)

## JavaScript Features
1. Contact form validation — validates all fields and email format
2. News live search — filters news cards as user types
