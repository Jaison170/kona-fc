# Kona FC Website

A multi-page responsive website for Kona FC, a fictional football club.
Built with HTML5, CSS3, JavaScript, and Bootstrap 5.

## Pages
- Home
- About
- Fixtures & Results
- News
- Membership
- Shop
- Contact
- Help

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
